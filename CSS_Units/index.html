Pixels (px)

The px unit is an absolute length. It defines fixed size regardless of screen or parent. Commonly used for buttons, icons, borders, etc.

em

Definition: Relative to the font-size of the parent element. Nesting compounds the size.

Parent (font-size: 20px)
Child (font-size: 2em → 40px)
Grandchild (font-size: 1.5em → 60px)

rem

Definition: Relative to the font-size of the root HTML element (default: 16px). Doesn't depend on parent.

Parent (font-size: 20px)
Child (font-size: 2rem → 32px)
Grandchild (font-size: 1.5rem → 24px)

vw

Definition: 1% of the viewport's width.

Example box styled using vw

vh

Definition: 1% of the viewport's height.

Example box styled using vh

vmin

Definition: 1% of the smaller side (viewport width or height).

Example box styled using vmin

vmax

Definition: 1% of the larger side (viewport width or height).

Example box styled using vmax

Percent (%)

Definition: Percentage values are relative to the parent element. For height, the parent must have a defined height.

Width: 50% of parent (300px) = 150px
Height: 50% of parent (300px) = 150px
Height: 50% — will not work! (no height in parent)
Outer (width: 500px)
Middle (width: 50% → 250px)
Inner (width: 50% of 250px → 125px)
